# OLD

Esta pasta concentra itens nao ativos, material historico e artefatos arquivados.

## Regra

- nada aqui deve ser tratado como fonte oficial atual sem nova validacao
- codigo ativo continua fora de `OLD/`
- quando um item sai do fluxo principal, ele deve ser preservado aqui em vez de ser apagado

## Grupos principais

- `00_INDICE_E_STATUS/` a `06_VISUAL_BOARDS_LINKS/`: trilha historica de prompts, packs e apoio visual
- `banco_legado/`: SQL, mapeamentos e documentos de modelagem antiga
- `arquivos_soltos/`: arquivos isolados sem papel ativo
- `materiais_visuais/`: imagens e apresentacoes de apoio
- `codigo_solto/`: codigo deslocado ou sem runtime valido
- `scripts/` e `tests/`: duplicidades antigas substituidas por `tooling/`
