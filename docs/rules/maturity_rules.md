# MATURITY RULES

## DIMENSÕES
- Diversificação
- Consistência de aporte
- Clareza de objetivo
- Controle de risco

## SCORE
Cada dimensão: 0–25
Total: 0–100

## CLASSIFICAÇÃO
- 0–40: iniciante
- 41–70: em evolução
- 71–100: estruturado

## REGRA
Maturidade NÃO depende de renda.
Depende de comportamento.
