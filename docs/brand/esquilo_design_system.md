# ESQUILO INVEST — DESIGN SYSTEM

---

## 1. FUNDAMENTOS

O sistema visual do Esquilo é baseado em três pilares:
- Clareza
- Leitura rápida
- Decisão

Nada é decorativo. Tudo precisa ajudar o usuário a entender e agir.

---

## 2. GRID E LAYOUT

### Grid base
- 8px base
- espaçamentos múltiplos de 8

### Containers
- padding horizontal: 16px (mobile)
- padding desktop: 24px

### Regra
Nunca usar espaçamento arbitrário.
