# 🐿️ Esquilo Invest
## Playbook de Implementação — Codex

---

## OBJETIVO

Definir execução modular e previsível do backend e frontend.

---

## REGRAS GLOBAIS

- seguir este documento sempre
- implementar um módulo por vez
- não inventar regras
- não usar IA para cálculo
- manter arquitetura modular

---

## EXECUÇÃO

1. identificar módulo
2. seguir seção do módulo
3. respeitar critérios de pronto

---

## PROMPT GLOBAL

Leia este documento antes de iniciar.
Implemente apenas o módulo solicitado.
Não invente lógica fora da documentação.
