# Exportação para PDF

## Arquivos principais

- `docs/esquilo_biblia_produto.html`
- `docs/esquilo_biblia_produto.css`
- `docs/assets/esquilo_logo.png`

## Como abrir

1. Abra `docs/esquilo_biblia_produto.html` em um navegador moderno.
2. Aguarde o carregamento completo do CSS e das imagens locais.

## Como exportar para PDF

1. Use a função de impressão do navegador.
2. Destino: `Salvar como PDF`.
3. Papel: `A4`.
4. Escala: `100%`.
5. Margens: padrão do documento.
6. Ative `gráficos de segundo plano`.
7. Desative cabeçalhos e rodapés automáticos do navegador.

## Observações

- O documento foi desenhado para leitura premium em tela e para exportação em A4.
- As variáveis e tokens visuais estão centralizados no CSS para customização futura.
- Se o navegador não encontrar as fontes locais esperadas, fará fallback tipográfico sem quebrar a estrutura.
