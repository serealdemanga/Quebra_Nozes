# MATURITY POINTS — REGRAS

---

## OBJETIVO

Medir a prontidão do usuário para trabalhar metas financeiras reais.

Não é gamificação.
É leitura de maturidade.

---

## ESCALA

Pontuação de 0 a 3.

- 0 → desorganizado
- 1 → básico
- 2 → consistente
- 3 → avançado

---

## CRITÉRIOS

### 1. Organização
- sem visão clara da própria carteira → 0
- visão parcial → 1
- visão clara → +1

### 2. Consistência de aporte
- inexistente → 0
- irregular → 1
- consistente → +1

### 3. Coerência da carteira
- incoerente com a realidade → 0
- média → 1
- coerente → +1

---

## RESULTADO

A soma é limitada a 3.

---

## USO

A pontuação define:
- se metas devem ser liberadas ou sugeridas
- qual o nível de complexidade das recomendações
- quão sofisticadas podem ser as projeções

---

## REGRA

- não usar como ranking
- não mostrar como sistema de pontos de jogo
- usar como leitura de prontidão

---

## REGRA FINAL

Se parecer gamificação infantil, está errado.
Se ajudar a personalizar o produto, está certo.
