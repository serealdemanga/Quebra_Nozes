# Placeholders de logo

## Objetivo
Garantir aparência consistente quando a logo oficial da instituição não estiver disponível.

## Estratégia
- retângulo com fundo neutro
- sigla ou iniciais em destaque
- mesmo espaçamento e mesma proporção para todas as instituições

## Exemplos de iniciais
- XP
- IT
- BB
- BR
- ST
- CX
- NU

## Regra
Placeholder não é gambiarra visual.
Ele faz parte do sistema de identidade da interface.
