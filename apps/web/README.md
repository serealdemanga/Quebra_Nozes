# App web

Esta pasta contém o front web do produto.

## App executável (Node)

> ✅ **Código ativo** — Vite + React + TS + React Router + Tailwind.

O app rodável vive em `apps/web/app`.

- Tipografia e tokens seguem o BrandBook (Sora/Inter + cores).
- Deve consumir contratos estáveis e não depender do legado em Apps Script.

## Estruturas auxiliares (não executáveis)

> ⚠️ As pastas abaixo são **artefatos de design e referência** — não são código integrado ao app.

| Pasta | Conteúdo |
|---|---|
| `src/` | Scaffolding inicial de features (home, portfolio, profile) — migrar para `app/` conforme as USs |
| `wireframes/` | Wireframes estáticos de referência de UX |
| `prototypes/` | Protótipos HTML/JS para validação de fluxos |
| `jsfigma/` | Exports do Figma para referência visual |

Essas pastas serão absorvidas gradualmente pelo app executável conforme as USs forem fechadas.
