# jsFigma estrutural do webapp

Esta pasta guarda artefatos estruturais do produto em JavaScript para orientar montagem no Figma sem virar design final.

Regra desta pasta:
- nao redefinir identidade visual
- nao polir interface
- nao tratar como codigo de runtime
- usar apenas como mapa executavel de shell, telas, navegacao, estados e componentes base

Fonte visual obrigatoria:
- `docs/brand/esquilo_design_system.md`
- `docs/brand/esquilo_brand_guide.md`
- `docs/brand/esquilo_ui_examples.md`

Arquivo principal:
- `esquilo_webapp_estrutural.js`

Uso esperado:
1. montar o shell do webapp
2. distribuir as telas e seus blocos principais
3. respeitar tipografia, cores e espacamentos ja definidos
4. manter o prototipo navegavel e estrutural
5. evitar qualquer acabamento visual fora da base documentada
